# 1_1 课程简介

## 核心概念

- 本课程为 C++ 编程语言教学
- 学生应具备 C 语言和 Python 编程基础
- 编程的精髓在于算法思想与工程实践，而非语言工具本身

## 原理理解

计算机专业核心课程体系包含四大支柱：

| 课程 | 作用 |
|------|------|
| **数据结构** | 直接影响编程实现能力 |
| **算法分析** | 决定程序效率的核心学科 |
| **操作系统** | 作为计算机资源管理者，控制所有程序的执行权限 |
| **计算机网络** | 支撑现代互联网应用的基础架构 |

## 总结

掌握编程核心在于算法与工程实践，而非语言本身。

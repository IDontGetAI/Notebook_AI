# 1_4 内存的逻辑理解

## 核心概念

- 内存可以看作一个巨大的数组
- 每个存储单元有唯一的地址
- 内存按字节（Byte）编址

## 原理理解

内存的逻辑结构：

- 内存是由**连续的存储单元**组成的线性空间
- 每个存储单元有唯一的**地址**（从 0 开始）
- 地址用十六进制表示（如 0x0000, 0x0001, ...）
- **1 Byte = 8 bits**，是内存的最小寻址单位

## 代码示例

```cpp
// 变量在内存中的存储
int a = 10;  // 假设地址为 0x1000，占用 4 字节
// 内存布局：[0x1000]=0A [0x1001]=00 [0x1002]=00 [0x1003]=00
```

## 易错点

- 不要混淆**变量的值**和**变量的地址**
- 不同数据类型占用的字节数不同

## 总结

理解内存的线性地址空间是指针学习的基础。

# 2_6 关系运算符

## 核心概念

- **关系运算符**用于比较两个值的大小关系
- 运算结果是**布尔值**：`true` (真) 或 `false` (假)

## 原理理解

关系运算符列表：

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | true |
| `!=` | 不等于 | `5 != 3` | true |
| `>` | 大于 | `5 > 3` | true |
| `<` | 小于 | `5 < 3` | false |
| `>=` | 大于等于 | `5 >= 5` | true |
| `<=` | 小于等于 | `5 <= 3` | false |

## 代码示例

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5;

    cout << "a == b: " << (a == b) << endl;  // 0 (false)
    cout << "a != b: " << (a != b) << endl;  // 1 (true)
    cout << "a > b: " << (a > b) << endl;    // 1 (true)
    cout << "a < b: " << (a < b) << endl;    // 0 (false)
    cout << "a >= 10: " << (a >= 10) << endl; // 1 (true)
    cout << "a <= b: " << (a <= b) << endl;  // 0 (false)

    return 0;
}
```

## 易错点

- **混淆 `=` 和 `==`**：`=` 是赋值，`==` 是比较
- **布尔值输出**：C++ 中 `true` 输出为 `1`，`false` 输出为 `0`

## 总结

关系运算符返回布尔值，常用于条件判断语句中，注意区分赋值运算符和等于运算符。

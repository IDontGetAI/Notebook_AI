#### 1.等于号

﻿

00:12

﻿

- 等于号是最基础的运算符，用于赋值操作。
- 类型转换规则：当double类型赋值给int类型时，会丢失小数部分。例如：int a = - 6，实际结果为a = 3。
- 整数除法规则：两个int类型相除，结果仍为int类型。例如：5 / 2结果为2，而非- 5。
- 取模运算：%运算符用于计算余数。例如：5 % 2结果为1。
- 混合类型运算规则：若运算涉及不同类型，结果会向更复杂的类型转换。例如：5 / - 0结果为double类型的- 5。

#### 2.加加操作符

﻿

07:31

﻿

|   |   |   |   |
|---|---|---|---|
|操作符位置|运算顺序|示例|结果|
|前置（++e）|先自增，后参与其他运算|int e=1; f=++e*2|e=2, f=4|
|后置（e++）|先参与其他运算，后自增|int g=1; h=g++*2|g=2, h=2|

核心规则：

- 前置自增会先改变变量值，再使用新值参与运算；
- 后置自增会先使用原值参与运算，再改变变量值。
- 类似运算符：减减（--）遵循相同规则。

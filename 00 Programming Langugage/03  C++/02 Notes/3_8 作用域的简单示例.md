# 3_8 作用域的简单示例

## 核心概念

- 通过具体示例理解作用域的行为
- 变量遮蔽（shadowing）的实际效果
- for 循环中的作用域特性

## 原理理解

### 示例 1：变量遮蔽

```cpp
int a = 10;  // 外层变量
{
    int a = 20;  // 内层变量，遮蔽外层
    cout << a;  // 输出 20
}
cout << a;  // 输出 10，外层变量未改变
```

### 示例 2：for 循环作用域

```cpp
// C++11 及以后
for (int i = 0; i < 5; i++) {
    // i 只在循环内可见
}
// cout << i;  // 错误！i 已离开作用域

// 旧式写法
int i;
for (i = 0; i < 5; i++) {
    // ...
}
cout << i;  // 正确，i 在循环外仍可见
```

## 代码示例

```cpp
#include <iostream>
using namespace std;

int x = 1;  // 全局变量

void func() {
    int x = 2;  // 局部变量，遮蔽全局变量
    cout << "func 内 x = " << x << endl;  // 输出 2
}

int main() {
    // 示例 1：函数内变量遮蔽
    func();
    cout << "main 内 x = " << x << endl;  // 输出 1

    // 示例 2：for 循环作用域
    for (int i = 0; i < 3; i++) {
        int j = i * 2;
        cout << "i = " << i << ", j = " << j << endl;
    }
    // cout << i;  // 错误：i 已离开作用域
    // cout << j;  // 错误：j 已离开作用域

    // 示例 3：嵌套作用域
    int value = 100;
    {
        int value = 200;  // 遮蔽外层
        {
            int value = 300;  // 再次遮蔽
            cout << "最内层 value = " << value << endl;  // 300
        }
        cout << "中间层 value = " << value << endl;  // 200
    }
    cout << "最外层 value = " << value << endl;  // 100

    return 0;
}
```

## 运行结果

```
func 内 x = 2
main 内 x = 1
i = 0, j = 0
i = 1, j = 2
i = 2, j = 4
最内层 value = 300
中间层 value = 200
最外层 value = 100
```

## 易错点

- **误以为内层改变了外层变量**：内层同名变量只是遮蔽，不改变外层
- **循环变量泄露**：for 循环内定义的变量在循环外不可访问
- **作用域嵌套过深**：过多的嵌套会降低代码可读性

## 总结

作用域示例展示了变量遮蔽的行为，内层作用域的同名变量不会影响外层，for 循环变量只在循环内有效。

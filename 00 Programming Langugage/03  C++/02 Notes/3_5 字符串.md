## 核心概念
- C语言字符数组被视为字符串时需以`'\0'`作为结束符
- 字符数组变量名本质是指向内存地址的指针
- C++的string类通过运算符重载实现内容比较功能
- 相同内容的字符数组可能存储在不同的内存地址

## 语法定义
```cpp
// C风格字符数组定义
char str1[] = "hello";  // 自动包含'\0'，实际长度为6
char str2[6] = {'h', 'e', 'l', 'l', 'o', '\0'};

// C++ string类定义
#include <string>
string s1 = "hello";
string s2 = "hello";

// 字符数组初始化（确保容量足够）
char str[10] = "hello";  // 容量需 >= 字符串长度+1
```

## 示例代码

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 示例1：C语言字符数组比较
    char a[] = "hello";  // 数组a
    char b[] = "hello";  // 数组b

    cout << "字符数组比较:" << endl;
    cout << "a == b (地址比较): " << (a == b ? "true" : "false") << endl;  // false
    cout << "a的地址: " << (void*)a << endl;
    cout << "b的地址: " << (void*)b << endl;
    // a和b存储于不同内存地址

    // 示例2：C++ string类比较
    string s1 = "hello";
    string s2 = "hello";

    cout << "\nstring类比较:" << endl;
    cout << "s1 == s2 (内容比较): " << (s1 == s2 ? "true" : "false") << endl;  // true
    // string类重载了==运算符，比较内容而非地址

    // 示例3：字符数组初始化注意事项
    char str[10] = "hello";  // 正确：容量10 > 长度5+1
    // char str[5] = "hello";  // 错误：容量5 < 需要6（含'\0'）

    cout << "\n字符数组内容:" << endl;
    cout << "str = " << str << endl;

    return 0;
}
```

## 运行结果 / 现象
```
字符数组比较:

a的地址: 0x7ffe...
b的地址: 0x7ffe...

string类比较:
s1 == s2 (内容比较): true

字符数组内容:
str = hello
```

## 原理理解
- 字符数组原理：
  - 字符数组a包含6个元素（含结束符`'\0'`）
  - 变量a与b的本质是内存地址指针
  - 比较运算符`==`实际比较的是指针指向的地址是否相同
  - 字符数组会独立分配空间，相同内容的数组地址不同
- 常量存储机制：
  - 常量存储在内存的常量区
  - 相同内容的常量字符串可能共享同一内存区域
  - 字符数组通过拷贝常量内容初始化，分配独立内存空间
- string类与运算符重载：
  - string类重载`==`运算符，比较逻辑为逐字符匹配直至结束符
  - 运算符重载的本质是根据操作数类型动态调整行为
  - 重载后的运算符比较字符串内容而非地址

## 易错点
- C语言字符数组用`==`比较比较的是地址，不是内容
- 字符数组初始化时要确保容量足够容纳字符串及结束符`'\0'`
- 数组长度小于字符串实际长度会导致越界错误
- C++的string类和C风格字符串有本质区别

## 面试 / 考试常见问法

- 什么是运算符重载？string类如何实现内容比较？

for循环结构包含四个关键部分：

- 初始化计数器（如int i = 0）
- 循环条件判断（如i < 3，条件为真则继续执行）
- 循环体代码执行 - 计数器更新（如i++）循环终止条件：当i = 3时，i < 3为假，循环结束。

do-while循环特点：

- 先执行循环体，再判断条件
- 至少执行一次循环体与while循环对比：while循环可能因初始条件不满足而一次都不执行。

- 初始值：a = 0，i = 3
- 执行过程：先执行a += i和i--，再判断i > 3为假
- 结果：a = 3

- 初始值：a = 0，i = 3
- 执行逻辑：直接判断i > 3为假，跳过循环体
- 结果：a = 0

#### 2.循环嵌套

﻿

14:12

﻿

嵌套循环执行逻辑：

- 外层循环：i取值0、1、2
- 内层循环：每次i循环时，j完整执行1、2、3
- 终止条件：外层i = 3时结束数学等价性：嵌套循环可理解为外层每次迭代时完整执行内层循环。

##### 1) 例题:两层循环计算

﻿

16:24

﻿

两层循环计算结果为18。具体计算过程如下：

- 初始值：a=0
- 外层循环：i取值0、1、2
- 内层循环：j取值1、2、3
- 运算逻辑：每次循环执行a += i * j（等价于a = a + i * j）

逐次累加过程：

- i=0时：
    - 0×1=0
    - 0×2=0
    - 0×3=0
    - 累计结果：a=0
- i=1时：
    - 1×1=1
    - 1×2=2
    - 1×3=3
    - 累计结果：a=0+1+2+3=6
- i=2时：
    - 2×1=2
    - 2×2=4
    - 2×3=6
    - 累计结果：a=6+2+4+6=18

数学表达：

两层循环可转化为双重求和公式：

﻿∑i=02∑j=13(i×j)\sum_{i=0}^{2} \sum_{j=1}^{3} (i \times j)i=0∑2​j=1∑3​(i×j)﻿

关键点：

- 循环层数定义：嵌套深度决定层数（本例为两层），与循环次数无关。
- 跨学科联系：编程中的循环与数学中的求和符号本质相通，但表述语言不同可能导致理解隔阂。

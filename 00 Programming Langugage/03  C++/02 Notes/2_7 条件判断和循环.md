# 2_7 条件判断和循环

## 核心概念

- **条件判断**：`if-else` 语句根据条件执行不同代码
- **循环**：`for`、`while`、`do-while` 用于重复执行代码

## 原理理解

### if-else 条件判断

```cpp
// 基本形式
if (条件) {
    // 条件为真时执行
} else {
    // 条件为假时执行
}

// 多分支
if (条件 1) {
    // ...
} else if (条件 2) {
    // ...
} else {
    // 默认情况
}
```

### 三种循环结构

| 循环类型 | 语法 | 特点 |
|----------|------|------|
| **for** | `for(初始; 条件; 更新)` | 先判断后执行，适合已知次数 |
| **while** | `while(条件)` | 先判断后执行，适合未知次数 |
| **do-while** | `do { ... } while(条件);` | 先执行后判断，至少执行一次 |

## 代码示例

```cpp
#include <iostream>
using namespace std;

int main() {
    // if-else 示例
    int score = 85;
    if (score >= 90) {
        cout << "优秀" << endl;
    } else if (score >= 60) {
        cout << "及格" << endl;
    } else {
        cout << "不及格" << endl;
    }

    // for 循环：输出 1-5
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    cout << endl;

    // while 循环
    int j = 1;
    while (j <= 5) {
        cout << j << " ";
        j++;
    }
    cout << endl;

    // do-while 循环
    int k = 1;
    do {
        cout << k << " ";
        k++;
    } while (k <= 5);

    return 0;
}
```

## 易错点

- **for 循环忘记更新**：导致死循环
- **while 条件永远为真**：导致死循环
- **do-while 结尾漏掉分号**：`while(条件);` 需要分号

## 总结

条件判断和循环是程序流程控制的基石，for 适合已知次数的循环，while 适合未知次数，do-while 保证至少执行一次。

# 1_6 编程语言的发展

## 核心概念

- **机器语言**：二进制代码，计算机直接执行
- **汇编语言**：助记符表示机器指令
- **高级语言**：接近自然语言，需要编译/解释

## 原理理解

编程语言发展史：

| 代 | 语言类型 | 特点 | 示例 |
|----|----------|------|------|
| 第一代 | 机器语言 | 0 和 1 组成的二进制代码 | 10110001 |
| 第二代 | 汇编语言 | 使用助记符 | MOV, ADD, JMP |
| 第三代 | 高级语言 | 接近自然语言 | C, C++, Java, Python |
| 第四代 | 面向问题语言 | 特定领域 | SQL |

## 易错点

- 高级语言需要**编译或解释**才能被计算机执行
- 语言越高级，离硬件越远，**抽象程度越高**

## 总结

编程语言从低级到高级发展，抽象程度越来越高。

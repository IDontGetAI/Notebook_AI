# 1_8 操作系统是计算机管理员

## 核心概念

- 操作系统是计算机资源的管理者
- 管理 CPU、内存、I/O 设备等资源
- 提供系统调用接口供程序使用

## 原理理解

操作系统的核心功能：

| 功能 | 描述 |
|------|------|
| **进程管理** | 管理程序的执行（创建、调度、终止） |
| **内存管理** | 分配和回收内存空间 |
| **文件管理** | 管理文件的存储和访问 |
| **设备管理** | 管理输入输出设备 |
| **系统调用** | 为应用程序提供访问硬件的接口 |

## 易错点

- 程序**不能直接访问硬件**，必须通过操作系统
- 操作系统控制程序的**执行权限**

## 总结

操作系统是程序与硬件之间的桥梁，管理所有资源。

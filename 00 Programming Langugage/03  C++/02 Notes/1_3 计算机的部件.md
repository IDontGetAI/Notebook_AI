# 1_3 计算机的部件

## 核心概念

- 计算机由硬件和软件两部分组成
- 硬件包括：CPU、内存、硬盘、输入输出设备
- CPU 是计算机的运算和控制核心

## 原理理解

计算机主要部件：

| 部件 | 功能 |
|------|------|
| **CPU（中央处理器）** | 执行指令，进行运算和控制 |
| **内存（RAM）** | 临时存储正在运行的程序和数据 |
| **硬盘（ROM/磁盘）** | 长期存储数据和程序 |
| **输入设备** | 键盘、鼠标等 |
| **输出设备** | 显示器、打印机等 |

## 易错点

- 内存是**易失性存储**，断电后数据丢失
- 硬盘是**非易失性存储**，数据长期保存

## 总结

理解计算机各部件的功能是理解程序运行的基础。

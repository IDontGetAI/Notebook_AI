### 一、递归函数

﻿

00:02﻿

递归函数（recursive function）是指通过调用自身实现循环功能的函数。递归（recursion）的核心在于函数自我调用和设置终止条件，二者缺一不可。递归逻辑可通过循环结构等价实现。

#### 1.递归函数求解连乘产品

﻿

00:28

﻿

- 连乘（阶乘）的循环实现：输入数值n（如n=5），通过循环结构计算n×(n-1)×...×1，需初始化累加变量sum=1，循环终止条件为n=0。
- 递归实现要点：
    - 自我调用：函数内部调用自身并传递参数n-1
    - 终止条件：当参数为1时返回1，避免无限递归
- 执行过程分析：以n=5为例
    - 5 → 4 → 3 → 2 → 1（触发终止条件）
    - 逐层返回计算结果：1×2×3×4×5

#### 2.结束

﻿

﻿

递归与循环的本质均为重复执行操作，但递归存在显著缺陷：

|   |   |   |
|---|---|---|
|对比维度|循环（迭代）|递归|
|内存占用|固定栈空间|随调用次数线性增长|
|适用场景|简单循环控制|树形结构遍历（如前/中/后序遍历）|
|风险点|无栈溢出风险|未设置终止条件会导致栈溢出（stack overflow）|

递归的栈空间机制：每次函数调用均创建独立栈帧，若递归深度过大（如百万级），将耗尽内存空间。阶乘递归的栈帧创建顺序为5→4→3→2→1，直至触发终止条件后逐层释放。

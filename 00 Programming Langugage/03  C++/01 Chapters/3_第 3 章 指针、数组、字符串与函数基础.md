## 本章重点
- **指针的四大特性**：理解指针的类型、值（地址）、地址、作用域
- **数组的原理与操作**：掌握一维数组、二维数组的定义、初始化和遍历
- **字符串处理**：理解 C 风格字符串和 string 类的区别与使用
- **函数基础**：掌握函数的定义、声明、参数传递和返回值
- **作用域规则**：理解局部变量、全局变量、块级作用域的生命周期
- **递归函数**：理解递归的原理、终止条件和栈帧机制

## 本章关联知识
- 指针是 C++ 的核心特性，是后续学习内存管理的基础
- 数组和字符串知识为学习 string 类和容器打基础
- 函数是理解后续函数重载、模板函数的基础
- 递归是理解树形结构遍历和分治算法的前提

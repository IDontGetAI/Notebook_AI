# 第 2 章 C++ 基础语法入门

## 上级导航

## 本章内容

### [[2_1 头文件 iostream]]
C++标准输入输出流头文件，提供`cout`（输出）和`cin`（输入）功能。`using namespace std`用于使用标准命名空间，解决标识符冲突问题。相比C语言的`stdio.h`，iostream更加面向对象，命名空间机制有效解决了大型项目中函数名冲突的问题。

### [[2_2 main 函数]]
程序执行的唯一入口点，操作系统通过调用main函数启动程序。标准形式为`int main() { ... return 0; }`。程序生命周期：main函数开始执行 → 程序运行 → main函数结束 → 程序终止。main函数的返回值表示程序执行状态，0通常表示成功。

### [[2_3 常量 Constant]]
常量分为字面常量和const常量。字面常量包括整型（`1`, `100`）、浮点型（`3.14`默认为double，`3.14f`为float）、字符型（`'x'`）、字符串（`"hello"`）。const常量使用`const`关键字声明，声明后不可修改。注意浮点常量默认类型为double，字符用单引号、字符串用双引号。

### [[2_4 变量的 4 大特性]]
变量具有名称、类型、值、地址四大特性。类型决定存储格式及空间大小（int占4字节）；地址是内存位置标识，由操作系统随机分配。32位系统地址占4字节，64位系统占8字节。`a[i]`等价于`*(a + i)`，是编译器提供的语法糖。理解变量的四个特性是掌握指针的基础。

### [[2_5 基本运算符]]
包括赋值运算符（`=`）、算术运算符（`+`, `-`, `*`, `/`, `%`）、自增自减运算符（`++`, `--`）。类型转换规则：double赋值给int会丢失小数部分；整数除法结果为整数（`5/2=2`）；混合类型运算向更复杂的类型转换。前置自增（`++e`）先自增再参与运算，后置自增（`e++`）先参与运算再自增。

### [[2_6 关系运算符]]
用于判断条件表达式的真假，包括`>`, `<`, `>=`, `<=`, `==`, `!=`。在C/C++中，0表示假，非0值均表示真。关系运算符返回整型：条件为假返回0，为真返回1。常见错误：误用赋值运算符`=`代替相等比较`==`。编程核心是保证代码可读性与可维护性。

### [[2_7 条件判断和循环]]
条件判断使用`if-else`语句；循环包括`for`、`while`、`do-while`三种形式。for循环结构：初始化 → 条件判断 → 循环体 → 计数器更新。while循环先判断条件再执行，do-while先执行再判断条件（至少执行一次）。循环可嵌套，嵌套循环的执行逻辑是外层每次迭代时完整执行内层循环。

### [[2_8 break 和 continue 关键字]]
break用于立即终止当前循环；continue用于跳过当前循环的剩余部分并进入下一次迭代。break直接跳出整个循环，continue只跳过本次循环体剩余代码。这两个关键字用于控制循环的执行流程，实现更灵活的循环控制逻辑。

### [[2_9 switch 语句]]
多分支选择结构，根据变量值跳转到对应case标签处执行。switch执行机制为跳转逻辑，break用于防止继续执行后续case语句（穿透效应）。default语句在所有case条件都不匹配时执行。实际编程中使用频率较低，但特定场景下仍有一定应用价值。

## 分类词汇

### 基础概念 (Fundamentals)

| 词汇 | 释义 | 例句 |
|------|------|------|
| variable | 变量 | `x = 5` assigns the value 5 to the variable x. |
| function | 函数 | Define a function using the `def` keyword. |
| parameter | 参数 | Pass arguments to the function parameters. |
| return | 返回 | The function returns a value. |
| loop | 循环 | Use a `for` loop to iterate over a list. |
| condition | 条件 | The `if` statement evaluates a condition. |

### 数据类型 (Data Types)

| 词汇 | 释义 | 例句 |
|------|------|------|
| integer | 整数 | An integer is a whole number. |
| string | 字符串 | A string is a sequence of characters. |
| list | 列表 | A list is an ordered collection of items. |
| dictionary | 字典 | A dictionary stores key-value pairs. |
| tuple | 元组 | A tuple is an immutable sequence. |
| set | 集合 | A set contains unique elements. |

### 面向对象 (Object-Oriented Programming)

| 词汇 | 释义 | 例句 |
|------|------|------|
| class | 类 | Define a class using the `class` keyword. |
| object | 对象 | An object is an instance of a class. |
| method | 方法 | A method is a function defined in a class. |
| attribute | 属性 | An attribute stores the object's state. |
| inheritance | 继承 | Inheritance allows a class to inherit from another. |
| polymorphism | 多态 | Polymorphism allows objects to be treated as instances of their parent class. |
| encapsulation | 封装 | Encapsulation hides the internal state of an object. |

### 常用短语 (Common Phrases)

| 短语 | 含义 | 使用场景 |
|------|------|----------|
| pass by value | 值传递 | When a copy of the value is passed to a function |
| pass by reference | 引用传递 | When a reference to the object is passed |
| raise an exception | 抛出异常 | When an error occurs during execution |
| handle errors | 处理错误 | Using try-except blocks |
| iterate over | 遍历 | Looping through a collection |
| instantiate a class | 实例化类 | Creating an object from a class |
